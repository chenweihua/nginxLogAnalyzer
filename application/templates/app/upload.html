Upload your access.log (.zip, .gz, .tar.gz ), it is possible to upload more than one log, and your log will be extract automatically.
For quickly parsing yours logs, application uses multithreading mode. N & A analysis software is a kind of web analytics software that parses a server log file from a web server, and based on the values contained in the log file, derives indicators about when, how, and by whom a web server is visited. Reports are generated immediately, but data extracted from the log files is stored in a postgres database, allowing various reports to be generated on demand.
